# FinMate - AI-Driven Financial Wellness System

An web application designed to simplify personal finance management by combining automation, insights, and shared expense handling in one platform.

---

##  Features

- Expense Tracking  
  Record and manage income and expenses  

- AI-Powered Categorization  
  Automatically categorize transactions using AI  

- Smart Financial Insights  
  Generate daily/weekly summaries of spending patterns  

- Saving Goals  
  Set targets and track progress visually  

- Recurring Payments  
  Manage subscriptions and regular expenses  

- Shared Expense Management  
  Split bills, track contributions, and settle balances  

- Budget Alerts  
  Get notified when spending exceeds limits  

- Visual Dashboards  
  Graphs and charts for better financial understanding
   
- Reports  
  Generate and export detailed financial reports, including transactions, summaries, and insights
